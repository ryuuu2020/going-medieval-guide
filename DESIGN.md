# Going Medieval Guide — Design System

## Theme: 古卷·暗影 (Ancient Scroll · Dark Monastery)
**Type**: Dark Parchment + Serif + Asymmetric (3-Colony Design)
**Vibe**: Medieval monastery library at night — warm gold accents on dark aged parchment.

| Token | Value | Usage |
|-------|-------|-------|
| `--color-bg` | `#1a1510` | Page background |
| `--color-bg-card` | `#242018` | Card/table header background |
| `--color-bg-elevated` | `#2d261b` | FAQ, elevated containers |
| `--color-text` | `#e0d4c0` | Body text |
| `--color-text-muted` | `#a89878` | Secondary text |
| `--color-accent` | `#d4943a` | Headings, links, primary buttons |

## Typography
- **Headings**: Cormorant Garamond, EB Garamond, Georgia (serif)
- **Body**: Lora, Source Serif 4, Georgia (serif)
- **UI Labels**: Inter (sans-serif)

## Key Components
- `.card` — Elevated card with irregular border-radius (4px 14px 4px 14px)
- `.aged-border` — Hand-drawn border simulation with inner shadow
- `.btn-primary` — Gold accent button, dark text
- `.hero-section` — Dark gradient hero banner
- `.wax-tag` — Inline accent tag (like a wax seal)
- `.faq-snippets` — Expandable FAQ details elements
- `.drop-cap` — Drop cap for article intros
- `.sidebar-block` — Left-bordered sidebar sections
